MVP – Sistema de Inteligência de Vendas

Objetivo do Projeto
Este projeto é um MVP de Inteligência de Vendas desenvolvido com foco em análise de dados, organização de informações comerciais e apoio à tomada de decisão.

O sistema consolida dados de vendas em um modelo dimensional (modelo estrela), permitindo análises por período, produto, cliente e vendedor.

---

Tecnologias Utilizadas
- SQL Server
- Python
- Power BI
- GitHub

---

Arquitetura de Dados
- Modelo Dimensional (Star Schema)
- Separação entre tabelas Fato e Dimensão
- Camada de dados preparada para BI e Analytics

---

Principais Análises
- Faturamento total
- Vendas por período
- Performance por vendedor
- Produtos mais vendidos
- Análise por região
